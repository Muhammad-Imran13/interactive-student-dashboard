# 🎓 Student Portal (JavaScript Assignment)

## 📌 Overview

This project is a simple **Student Portal Web Application** developed using **HTML, CSS, and JavaScript**. It demonstrates fundamental web development concepts including layout design, navigation, form handling, and table display.

---

## 🚀 Features

* 🧭 **Sticky Navigation Bar**

  * Smooth scrolling between sections
  * Clean and modern UI

* 📝 **Student Input Form**

  * Collects student data
  * Styled input fields and dropdowns

* 📊 **Data Display Table**

  * Structured presentation of information
  * Clean and readable layout

* 🎨 **Modern UI Design**

  * CSS variables for theming
  * Responsive and user-friendly interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Variables)
* JavaScript (Basic DOM Manipulation)
